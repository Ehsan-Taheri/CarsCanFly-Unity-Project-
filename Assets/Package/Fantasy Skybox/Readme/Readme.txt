Fantasy Skybox 1.2.1
- - -- - - -  - - - -  - - -- - --  

	Brighten your world with colourful handpainted skyboxes.
	
	This package contains materials, textures, demo scene and script which helps you quickly add a colourful skybox to your scene. It is easy to use and modify.

	Features:
	
		• Twenty eight skyboxes from 7 different designs.
		• 1024x1024 texture for each side of skybox.
		• Sunny/night skyboxes.
		• Demo scene and script.
		• Light & render settings guide for each skybox.
		• Mobile friendly.
		
	Unity and Unity Pro compatible
		
	Product page:
	
			http://www.ge-team.com/pages/unity-3d/first-sky-sunny-v-1-0/
